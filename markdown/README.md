# AES PDF Protector

##  English

This is a simple command-line tool that encrypts PDF files using AES 128-bit encryption via PyPDF2.  
It helps protect documents from unauthorized access and copying.

### Features
- AES encryption (128-bit)
- Password protection
- CLI interface
- Easy to use

### Requirements
- Python 3.11+
- PyPDF2

### Usage

```bash
python protect.py
